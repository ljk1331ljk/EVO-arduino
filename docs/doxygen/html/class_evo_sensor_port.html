◆ begin()

bool EvoSensorPort::begin ( int retries = 9)

Starts the EV3 sensor communication protocol. First reads the sensor configuration and calls the onSuccess callback afterwards. Then keeps the connection online by sending the sensor NACK regularly. The method performs all commuincation in its own FreeRTOS task, there it returns immediately.
